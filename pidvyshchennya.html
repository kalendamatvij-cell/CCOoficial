Система підвищення НК

Рекрут [1] - Солдат [2]

  • • Здати Статут
  • • Здати Субординація
  • • Здати Присяга
  • • Здати Графік роботи
  • • Отримати роль у Discord

Солдат [2] - Старший солдат [3]

  • • Стройовий Статут
  • • Коди та Тен-Коди
  • • Система паркування
  • • Статут гарнізонної та вартової служб ЗСУ
  • • Виконати завдання "Склад озброєння" 2 рази

Старший солдат [3] - Капрал [4]

  • • Здати академії вашого підрозділу
  • • Виконати завдання "Паливо" 3 рази
  • • Виконати завдання "Чищення озброєння" 1 раз

Система підвищення ССО

Капрал [4] - Сержант [5]

  • • Виконати завдання "Наряд на АКПП" 2 рази
  • • Несення варти на бункері: 2 рази по 20 хв.
  • • Несення варти на Вежі: 2 рази по 20 хв.
  • • Патруль навколо ВЧ: 2 раз - 20 хв.
  • • Виконати завдання "Морське патрулювання" 1 раз
  • • Участь у тренуванні: 3 рази (для бійця). Для офіцера та вище - провести 5 тренувань
  • • Допомога на наборі або виїзд на БП: 1 раз

Сержант [5] - Старшина [6]

  • • Допомога на наборі або виїзд на БП: 2 рази
  • • Несення варти на бункері: 3 раз по 20хв.
  • • Патруль навколо ВЧ: 2 раза по 20хв.
  • • Патруль по містах: 2 раз - 20хв.
  • • Виконати завдання "Наряд на АКПП" 2 рази
  • • Виконати завдання "Морське патрулювання" 2 рази
  • • Участь у РП-тренуваннях/школіннях: 5 разів (для бійця). Для офіцера та вище - Провести 4 РП-тренування/школіння
  • • Виконати 1 Спец.завдання

Старшина [6] - Лейтенант [7]

  • • Допомога на наборі або виїзд на БП: 3 рази
  • • Несення варти на бункері: 3 раз по 20хв.
  • • Виконати завдання "Морське патрулювання" 2 рази
  • • Виконати 1 Спец.Завдання
  • • Патруль навколо ВЧ: 3 раза по 20хв.
  • • Патруль по містах: 3 раз - 20хв.
  • • Виконати завдання "Наряд на АКПП" 4 рази
  • • Привезти 150 Ящиків БК (рахується вся к-сть розгружена на ВЧ)
  • • Взяти участь у тренуваннях 4 рази. Для офіцера та вище - Провести 6 тренувань
  • • Прослухати 3 лекції. Для офіцера та вище - провести 5 лекцій

Лейтенант [7] - Капітан [8]

  • • Мати посаду офіцера, або више
  • • Допомоги на наборі або виїзд на БП: 4 раз
  • • Несення варти на бункері: 3 рази по 20хв.
  • • Виконати завдання "Морське патрулювання" 2 рази
  • • Провести 8 тренувань та 6 лекцій
  • • Патруль навколо ВЧ (Спартан або Toyota Hilux): 4 раз по 20 хв.
  • • Патруль міста: 3 раз - 20 хв.
  • • Виконати завдання "Наряд на АКПП" 5 разів
  • • Привезти 200 Ящиків БК (рахується вся к-сть розгружена на ВЧ)
  • • 1 спец. Завдання
  • • Повторно здати Статут і правила ССО
